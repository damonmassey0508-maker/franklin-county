5-step AI-powered research process — Franklin County

1
Seed Data Pull — Franklin County Auditor
Download the full parcel roll for Columbus, Whitehall, Reynoldsburg, Bexley, and suburbs. Filter for: delinquent taxes 2+ years, vacant classification, last transfer 5+ years ago, or absentee ownership (owner address ≠ property address).
Tool: franklincountyauditor.com
2
Code Violation Cross-Reference
Submit FOIA requests to Columbus Division of Code Enforcement and suburb building departments. Request open violation logs, vacant building registrations, and boarded-up property lists. Properties on 2+ lists are highest priority.
Tool: Columbus Code Enforcement + suburb FOIA requests
3
AI Scoring with Claude
Upload your combined CSV to Claude. Score each property by distress indicators: violations + delinquent taxes + vacancy + absentee + POS status. Identify top 20–30 acquisition targets ranked by distress score.
Tool: Claude AI → claude.ai
4
COCIC Land Bank & POS Check
Cross-reference your scored list against COCIC's property inventory. For Bexley properties: pull failed POS inspections from Building Dept. Land bank and POS failures are highest-motivation sellers.
Tool: COCIC + Bexley Building Dept
5
Skip Trace, Outreach & Validation
Run skip traces on absentee and LLC owners via BatchLeads. Use Claude to draft personalized outreach letters. Drive-by validate top 30 properties — photograph boarding, utility status, and condition indicators.
Tool: BatchLeads + Claude AI + field visit
